# Vue.js 仿 CNode 技术社区

## 介绍
通过 [CNode开源技术社区](https://cnodejs.org/) 提供的API，实现的一个仿 CNode 社区的 Vue.js 作品

## 预览
- [Vue.js仿CNode技术社区](https://zhuhuang.me/vue-cnode-demo/dist/#/)
- [官方API地址](https://cnodejs.org/api)

# 实现功能
- 文章列表
- 文章分类
- 文章详情
- 用户详情
- 适配移动端

## 技术栈
- Vue.js
- Vue Router
- Axios
- HTML5
- CSS3
